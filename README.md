# FitTrack 3D 🏋️‍♂️

Geleneksel spor programı takip uygulamalarına yeni bir soluk! React, Three.js ve TailwindCSS kullanarak geliştirdiğim bu projede, fütüristik bir arayüz, glassmorphism kartlar ve interaktif 3D arka planlarla sıkıcı antrenman günlüklerini sonlandırıyoruz.

Sunucuya ihtiyaç duymadan, %100 yerel (localStorage) ve JSON yedekleme desteğiyle tüm verileriniz güvende.

Sınırları zorla, gelişimini takip et!
